# Media Capture Guide

Use this folder for real project media shown in the README.

Recommended files from real, non-sensitive project runs:

- `demo.gif` — 10-20 second walkthrough: choose folder, dry-run, organize, search.
- `cli.png` — terminal dry-run or successful run.
- `desktop-gui.png` — Tkinter result table with categories.
- `streamlit.png` — upload/classification view.

Do not commit generated fake demo folders, staged sample documents, screenshots
containing private filenames, API keys, local paths, or personal documents.
