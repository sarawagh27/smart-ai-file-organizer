# Roadmap

This roadmap keeps the project focused on practical file organization rather
than platform sprawl.

## Near Term

- Add real screenshot assets for the CLI and desktop GUI.
- Add optional coverage publishing once the repository has a public coverage target.
- Extract UI and watch-mode business logic into testable adapters.
- Add manual smoke-test evidence for CLI, GUI, and Streamlit demo flows.

## Later

- Package a desktop executable for non-technical users.
- Add drag-and-drop desktop workflows.
- Add a local-first embedding cache with clear rebuild controls.
- Add import/export for category presets.

## Non-goals

- Cloud storage sync.
- Background upload of private documents.
- Replacing user review with fully autonomous live moves.
